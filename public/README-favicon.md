# 🎨 Favicon du Portfolio

## Design 

La favicon utilise les initiales **"SL"** (Sohaib LAARICHI) avec un design moderne et professionnel.

### Caractéristiques :
- **Couleurs principales** : Dégradé cyan (#0ea5e9) vers violet (#6366f1)
- **Style** : Coins arrondis, ombres subtiles, éléments décoratifs
- **Typo** : Arial Bold pour une excellente lisibilité
- **Format** : SVG pour une qualité parfaite à toutes les tailles

## Fichiers inclus

- `favicon.svg` - Favicon principale (32x32, mode clair)
- `favicon-dark.svg` - Version mode sombre
- `favicon-16x16.svg` - Version petite taille
- `apple-touch-icon.svg` - Icône Apple iOS (180x180)
- `manifest.json` - Configuration PWA

## Compatibilité

- ✅ Navigateurs modernes (Chrome, Firefox, Safari, Edge)
- ✅ Mode sombre/clair automatique
- ✅ iOS/Android (PWA ready)
- ✅ Toutes les résolutions (SVG vectoriel)

## 🎯 Pourquoi cette design ?

1. **Professionnalisme** : Couleurs tech modernes
2. **Reconnaissance** : Initiales claires et mémorables  
3. **Cohérence** : S'harmonise avec le thème du portfolio
4. **Performance** : SVG léger et scalable
